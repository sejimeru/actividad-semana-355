FALLOS EN C++

hay diferentes tipos de error en un algoritmo de c++, algunos de ellos son:
1. NO CIERRA CON PUNTO Y COMA(;) : este error suele pasar y es que en un algoritmo de c++ nunca puede faltar un punto y coma para terminar cualquier variable
2. FALTAN LETRAS EN VARIABLES : es que no pueden faltar ninguna letra para poder que una variable funcione bien
3. UBICAR MAL ALGUNOS SIGNOS : para que un algoritmo funcione de manera correcta hay que saber ubicar los elementos bien ya que si no lo hacemos el algoritmo no funcionara

hay 4 instrucciones en las que tambien pueden haber fallos:
1. IMPRESION: la mas conocida de esta es cuando fallamos en el "cout" que es la forma de impresion, aveces fallamos en el mayor y menor
2. PETICION: la mas conocida de la instruccion es el "cin", fallamos cuando no tenemos variable y cuando no hay punto y coma
3. DECLARACION: la mas usada es "int", fallamos cuando tenemos errores de escritura, o cuando no ponemos valor a lo que necesitamos
4. MODIFICACION: esta no es tan utilizada pero de igual manera tiene errores, como por ejemplo poner de forma equivovada la modificacion
